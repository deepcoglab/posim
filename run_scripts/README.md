# Run Script Samples

This directory contains public sample scenarios for POSIM.

Each event folder includes:

- `config.json`: a sanitized configuration template with placeholder API credentials
- `data/users.json`: sample user profiles
- `data/events.json`: sample exogenous events
- `data/initial_posts.json`: sample seed posts
- `data/relations.json`: sample social relations

The JSON files under `data/` are intentionally small samples. They are not the full datasets used in experiments.
